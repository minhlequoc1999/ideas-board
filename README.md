# Ideas — capture board

A fast, no-friction place to jot down random ideas and todos. Open it from any
device via its URL.

## Features
- **Quick capture** — type an idea, hit Enter, done.
- **Drag to reorder** — grab the handle (⠿) to push important things to the top. Works on touch too.
- **Pin / edit / delete / complete** — every item.
- **Deadlines** — set a date per item; an *Upcoming deadlines* panel shows what's due (and what's overdue).
- **Tags with custom colors** — define your own tags, color them, assign them, and filter by them.
- **Import from a `.txt` / `.md` file** — bullets, numbered lists, and `- [ ]` checkboxes become tasks. Markdown headings (`#`) become tags.
- **Search** — press `/` to jump to search.
- **Light / dark theme.**
- **Export / Import backup** — `.json` file to move your data to another device.

## Where is my data?
Everything is stored in your browser (`localStorage`) on the device you're using.
It is **not** synced across devices automatically. To move data, use **Export**
on one device and **Import** on another.

## Hosting
This is a single static `index.html` — served via GitHub Pages.
